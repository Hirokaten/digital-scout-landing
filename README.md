## 🛠️ Про проект

**Digital Scout** — це Telegram-бот, який працює на випередження. Поки інші оновлюють сторінки вручну, Скаут аналізує ринок 24/7 і миттєво сповіщає про появу вигідних пропозицій.

### Основні фічі:
* **Автоматичне виявлення DEAL**: Алгоритм порівнює ціну лота з середньою по ринку і маркує вигідні варіанти позначкою 🔥.
* **Відстеження зниження ціни (DROP)**: Бот пам'ятає ціни і сповіщає, якщо продавець вирішив «посунутися».
* **Адаптивне сканування**: Інтервали перевірки підлаштовуються під активність ринку та час доби.
* **Мультивалютна оплата**: Підтримка Monobank та Telegram Stars.
